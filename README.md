
# Extra-Hours

Este proyecto tiene como objetivo automatizar la creación de formatos de reporte de horas extra para nómina, optimizando el proceso y reduciendo los errores manuales tanto para los gerentes de ingeniería como para el equipo de nómina. La solución incluye el desarrollo de un módulo frontend que permitirá a los gerentes ingresar y revisar las horas extra, y un módulo backend que gestionará los cálculos y almacenamiento de los datos, así como la generación de archivos en formatos Excel/CSV para su posterior procesamiento.

## Tech Stack

**Client:** React, JavaScript, Ant Design, Sass, Axios, Vite.

**Server:** Java, SpringBoot, MySQL, Hibernate, JWT.


## Installation

Primero, clona el repositorio desde GitHub:

### Backend

```bash
[https://github.com/Vane789/extra-hours-app]
cd backend 
```

#### Instala las dependencias: gradlew build
```bash
./gradlew build
gradlew build
```

#### Iniciar el servidor:
```bash
./gradlew bootRun
gradlew bootRun
```

### Frontend
```bash
cd frontend 
npm install o npm i 
```

#### Iniciar la aplicación:
```bash
npm run dev
```
